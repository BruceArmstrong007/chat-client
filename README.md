
# chat-client
Client Side of Chat Web App

# Features
Authentication - Register & Login
Send & recieve friend request
Chat with Friends
Chat will not be erased if u login later

# Hosted Web App
https://chat-ab2aa.web.app/

# Tech Stack
Angular,
Angular Material,
NGRX,
RxJs,
Socket.IO
SQLlite3


# Deployment - CI/CD Integration
